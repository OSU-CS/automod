AutoMod
=======


### To run locally:
* Install python3.7
* Create base venv (virtual environment)
    ```bash
    pip install virtualenv
    virtualenv venv
    ```
* Activate venv
    ```bash
    source venv/bin/activate
    ```
* Rebuild venv with dependencies
    ```bash
    pip install -r requirements.txt
    ```
* Run app
    ```bash
    python app.py
    ```
